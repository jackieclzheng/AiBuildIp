# 短视频文案规则（必遵守）

撰写任意短视频文案时，必须显式检查并体现以下四要素：

1) 预测（反常识开篇）
- 挑出大众认知盲点，提出反常识的“将要发生什么”，打破既有认知，让观众带疑问看完。

2) 奖励（奖励预期，多巴胺）
- 给出明确的收益/好处预期（更快赚钱、更省力、独家资源），让人脑提前“尝到甜头”，驱动完播与互动。

3) 损失（损失厌恶）
- 点出不行动的代价，“划走就损失机会/时间/金钱”，用损失厌恶促使立即关注、收藏、行动。

4) 命名（概念标签）
- 为观点/方法/模型命名，创造易记的概念（如“心力”），让内容有梗、易传播、便于引用与复用。

执行提醒：
- 每条文案输出前，自检是否覆盖并清晰表达以上四要素；缺一则补充。
- 示例话术可参考 `短视频文案原则.md`，但应结合当期主题、场景和受众做定制化改写。

检查清单（逐条打勾）：
- [ ] 开头有反常识/认知冲突的“预测”点题。
- [ ] 清晰的“奖励预期”已明示（具体收益/好处、可获得的资源）。
- [ ] “损失”后果明确，能触发损失厌恶（错过机会/时间/金钱）。
- [ ] 有概念“命名”或标签，便于记忆和传播。
- [ ] 话术已结合本次主题和受众做定制化，而非套话。
