# AI 内容产品矩阵（课程输出件）

## 1. 课程思路拆解
- **自有 IP 可信度**：来自《来时路.md》的履历（Autodesk 全球产品、Mate 系列兼容性负责人、10 年创业累计 35 所高校合作、AI 课程 5000 份销量），决定课程必须以“实战案例 + 真实交付日志”切入，而不是空谈理论。
- **选题即产品入口**：结合《爆款短视频选题_100.csv》的 100 个热点主题，课程的核心产物不是单纯讲解，而是“可直接发布的文案 + 脚本模板”，以此降低学员试错成本。
- **商业洞察差异化**：10 年创业经历带来的商业认知（投资人撤资、合伙人管理、跨行业交付）被拆成“选题判断模型 + 收益模拟表”，作为课程专属差异点，帮助学员不仅会写，还能算投入回收。
- **分层价值闭环**：课程要求至少覆盖“快速感知 → 系统搭建 → 定制交付”三个层级，因此以 99 / 399 / 1499 形成清晰的价值梯度，并与高客单的定制项目（来时路中的海外客户、医疗/教育软件项目）建立向上转化路径。

## 2. 客户画像 & 痛点
- **初级创作者 / 运营**：迫切需要稳定的热点灵感和圈粉话术，却没能力搭建智能体或内容系统。
- **中层内容经理 / 机构老师**：手里有课程或服务，但缺少“AI + 智能体”卖点和可复制流程。
- **AI / SaaS 创业者**：希望把智能体技术、行业案例包装成高客单产品，需要资深交付经验背书。

## 4. “输出件”说明
1. **灵感库（99 元交付）**：以 `hot-video-topics-copywriting.md` 为基础，自动生成每日推送（已经由 send_hot_topics_digest.py 实现轮播），学员购买后可立即获得 30 组可复制文案。
2. **脚本作坊作业簿（399 元）**：参考 `project-copywriting-1101V3.5.md` 的结构，为学员提供“标题 → 朋友圈 → 小红书”三段式模板，并附上来自《来时路.md》的真实故事示例，让学员知道如何把自身经历植入选题。
3. **智能体内容工厂（1499 元）**：复用当前仓库的自动邮件、脚本工具，交付一套完整的运营 SOP（每日选题 → 邮件播报 → 社群分享），并提供 1 对 1 的行业化改造建议。

## 5. 增长与交付策略
- **入口内容**：利用 `send_hot_topics_digest.py` 9:30 邮件，把 3 组爆款选题作为“免费体验”，附上 99 元灵感盒的购买链接。
- **信任构建**：用《来时路.md》里 Autodsk / 华为 / 创业 10 年故事做短视频，用 CSV 中的相关选题（例如“AI安全守护指南”“智能体如何帮助医疗行业提升效率”）做系列内容。
- **成交路径**：99 元购买页嵌入“加群”动作 → 群内 3 天内推介 399 元作坊；作坊结束后一对一邀约 1499 元陪跑，附赠真实项目案例或客户感谢信截图。

## 6. 后续可迭代方向
1. **话题矩阵**：继续扩充《爆款短视频选题_100.csv》，标注热度与适配产品档位，方便不同价位的内容快速调用。
2. **案例资产**：把《来时路.md》拆成图文 + 视频素材，为 399/1499 档位提供更多可引用的“信任片段”。
3. **工具打包**：将自动邮件脚本（send_hot_topics_digest.py、send_copywriting_digest.py）整理成轻量开源工具，作为 1499 元服务的“技术护城河”。
