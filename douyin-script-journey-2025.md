# 抖音视频脚本｜从研发工程师到 Agent 开发者的来时路

## 视频定位
- 受众：关注 AI 创业、自我成长的职场人、高校师生
- 时长：8 分钟
- 呈现方式：第一人称讲述 + 年代时间线字幕 + 过往照片/证书/参会视频 + 企业导师授课片段 + 日常 B-roll

## 开场（0:00-0:45）
- 画面：镜头对准自己或工作台，桌上放第一份工作留下的纪念品、跑鞋、咖啡杯、正在运行的代码界面；字幕出现“来时路｜2006-2025”
- 文案：  
  “大家好，我是 06 年毕业的老软件人，经历过全球软件研发、华为终端黄金年代、自主创业的修罗场，如今在做 Agent 开发。今天想用 8 分钟聊聊我的来时路，聊聊一个技术人如何在不同阶段重建自己。”

## 第一章：上海的全周期 baptism（0:45-2:00）
- 画面：Autodesk Inventor Tooling 项目的界面截图、旧工牌、产品全球发布会新闻、团队合影
- 文案要点：
  - “大学毕业后我去了上海一家美资企业，做 Autodesk Inventor Tooling。”
  - “那是罕见的本土团队主导全球软件，我们从市场调研、用户访谈开始，到架构、开发、测试，再到全球发布，全程都在上海完成。”
  - “当时国内很多软件团队还在做外包，我们却有机会亲手搭建产品，这让我第一次建立‘产品思维’：不是交差，而是交付价值。”
  - “我记得那个项目上线后，海外用户的反馈邮件写着‘finally a tooling solution built by the people who actually get manufacturing pain points’，那句话我保存到现在。”
- 过渡语：  
  “也就是在这个项目里，我第一次意识到：技术可以承载市场、管理和文化的多重要求。它为后面的一切打了底。”

## 第二章：华为的全球拉练（2:00-3:30）
- 画面：华为办公环境、海外差旅照片、UPF 与 CE4A 峰会资料、飞机舷窗、会议现场速记
- 文案要点：
  - “2010 年我加入华为终端，那时我们还在憋大招，想在 3000 元以上的高端机市场和苹果三星正面杠。”
  - “我的任务是解决手机与豪华车载系统的兼容性，这件事只能走出去。我几乎提着电脑住在飞机上：斯德哥尔摩、哥德堡、慕尼黑、维也纳、曼谷、洛杉矶……”
  - “我们向全球合作伙伴解释华为的能力，也吸收最前沿的行业标准细节。UPF 半年会、奔驰奥迪宝马大众保时捷联合的 CE4A 峰会，都有我们。”
  - “后来华为终端在一次舆情回应里，公开提到我们在蓝牙标准上的贡献。看到那条微博的瞬间，我在屏幕前默默笑了：原来那段奔波真有回响。”
- 过渡语：  
  “技术人的浪漫，也许就是在公司巨轮的日志里，留下自己那行不起眼的 commit。”

## 第三章：Mate7 之前与之后（3:30-4:30）
- 画面：华为 Mate 系列宣传片段、当年发布会片段、Mate7 指纹识别特写、排队购买的新闻素材
- 文案要点：
  - “Mate7 发布是整个团队的一个高光点。”
  - “当时我们内部反复想：高端机到底怎么样才算‘高端体验’？除了参数，还要稳定，又要好看，还要兼容汽车、耳机、外设。”
  - “指纹识别大获成功，第一批发货超过 100 万部。那天在街上看见陌生人用我们做过的手机，我有点想冲上去告诉他——嘿，这机器里有我的几行代码。”
  - “也就是在这里，我第一次真正理解品牌的力量，理解工程师的成就感是什么。”
- 过渡语：  
  “可是，人生不会只往上；所有上升之后，都可能是新的抉择。”

## 第四章：创业十年的修罗场（4:30-6:10）
- 画面：创业时期的办公室、团队合影、产品 Demo、客户现场、合作高校授牌仪式
- 文案要点：
  - “2015 年我在‘大众创业、万众创新’的大背景下离开华为，决定自己搭一艘小船。”
  - “一开始做 IT 培训和校企合作，后来扩展到软件定制服务。教育、医疗、先进制造等行业都踩过。”
  - “创业的路比想象中更崎岖：投资人撤资、合伙人撂挑子、职业经理人拉垮、团队内讧……领导当初挽留我时说‘华为给你的，是一个平台；你自己出来要从零搭平台’。后来我才彻底懂这句话。”
  - “但也正是这些坑，把组织能力和客户关系磨出来。最终我们建立了和 35 所高校的合作，落地了多个企业项目，还拿到了两家高新技术企业资质。”
- 过渡语：  
  “这十年像是在跑一个超长程马拉松，每次以为到终点了，其实只是补给站。”

## 第五章：危机、回馈与坚持（6:10-7:10）
- 画面：疫情期间远程办公镜头、武汉与交大捐款记录、团队线上会议截屏、跑步与咖啡的生活片段
- 文案要点：
  - “疫情是最大的黑天鹅。很多项目突然被叫停，团队也陷入迷茫。但技术人最怕失去手感，所以我把时间切成几个战场。”
  - “一边带团队攻克客户远程需求，另一边策划线上课程，把经验搬到腾讯课堂，机器学习课程单品卖出 5000 份，冲到第一。”
  - “我们还组织了两届全国数据分析大会，筹集资源为武汉、交大捐款。我想告诉团队：即使困难，也不要忘记回馈。”
  - “跑步和手冲咖啡，是那段时间的仪式感。跑圈里的耐心、咖啡里的专注，让我撑过那些失眠夜。”
- 过渡语：  
  “很奇怪，越是焦虑的时候，越要给自己一个节奏，让脚步稳下来。”

## 第六章：AI 的嗅觉与跨国磨炼（7:10-7:40）
- 画面：AI 课程研发现场、全国数据分析大会现场、与美国客户的视频会议、客户感谢信截图
- 文案要点：
  - “2019 年开始我对 AI 的敏感度被激活，带队研发课程、做企业咨询。”
  - “和美国客户合作定制项目，需求从英文书面到口头沟通全覆盖。上线那天客户写来长信，说我们的‘communication is crisp’。我在办公室差点原地起跳。”
  - “这些经历告诉我，技术不是一成不变的标签，它要随着市场变化不断刷新。”
- 过渡语：  
  “也正是那时，我开始思考：下一次浪潮是什么？”

## 第七章：重返一线的 Agent 之路（7:40-8:00）
- 画面：当前 Agent 项目界面、代码演示、团队研讨画面、内容创作流程
- 文案：  
  “现在我专注在 Agent 开发，把研发、产品、教学的经验融合，让企业能真正落地 AI 助手。我的来时路还在延伸，也欢迎你在评论区和我分享你的故事。我们一起看看，下一段旅程会把我们带到哪里。”

## 拍摄小贴士
- 构图：第一人称讲述部分尽量使用三分法，背景可放书架、证书或电脑屏幕；回忆段落可以使用慢摇镜头或推拉光效。
- 节奏：每章开头使用时间轴字幕（例如“2010 华为终端｜全球拉练”），配合轻微快门音效强调阶段切换。
- 音乐：选用具有成长感的电子氛围或轻摇滚伴奏，章节间可使用环境声（飞机广播、键盘声、咖啡滴滤声）增强沉浸感。
- 画面处理：历史照片可加轻微胶片颗粒，当前项目画面可采用屏幕录制 + 叠加关键数字动画。
- 互动：结尾呼吁观众评论，屏幕右下角浮现“留言分享你的来时路”提示。
